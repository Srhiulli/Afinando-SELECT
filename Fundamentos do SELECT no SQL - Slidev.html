O que é um banco de dados SQL?

Um banco de dados SQL é muito parecido com uma planilha do Excel. Ele guarda as informações em tabelas.

Exemplo de tabela de usuários:

idnameemailpassword
1Mariamaria@email.com12345
2Joãojoao@email.comabcdef
3Anaana@email.comsenha789

Qual a diferença para o Excel?

A diferença é que, em um banco de dados, usamos códigos para consultar e modificar os dados, e não uma interface gráfica.

Esses códigos são chamados de queries.

Clicks Animations

You can add v-click to elements to add a click animation.

This shows up when you click the slide:

<div v-click>This shows up when you click the slide.</div>
